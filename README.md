## Featured Projects

### 3CX Nexus

A multi-tenant VoIP management platform for handling multiple 3CX instances from one place.

**Main work:**

- Built a React and TypeScript dashboard for managing live 3CX instances.
- Created backend API routes with Node.js and Express.
- Used MongoDB for persistence and Redis-style caching concepts to reduce repeated external API requests.
- Improved data loading and reduced unnecessary API usage through caching and query optimization.

**Stack:** React, TypeScript, Redux Toolkit, Node.js, Express, MongoDB, Docker, Redis

---

### Competitive Arena Combat Prototype

A Unity-based top-down arena combat prototype focused on movement, combat clarity, and enemy wave systems.

**Main work:**

- Implemented player movement, mouse aiming, dash cooldowns, hit feedback, and invulnerability frames.
- Built basic projectile combat and enemy AI behavior.
- Added wave spawning, health systems, UI feedback, and gameplay balancing.
- Focused on readable game-feel systems instead of only visual polish.

**Stack:** Unity, C#, gameplay systems, enemy AI, UI systems
